title: Markdown 语法说明

简述

  • 理念

    让文档易写易读

  • 元素

    • 段落

      段落是由一个或多个连续的文本行组成，他的前后要有一个以上的空行

    • 换行

      单个回车是空格，连续回车是分段。

    • 标题

      • 最高标题（用底线====）

      • 第二级标题(用底线—–)

      • 其他标题(在行首加不同数量的#号表示不同的标题)

    • 引用

      在每一行的最前面加上右箭头(>)表示引用。

      引用还可以嵌套用不同数量的>就可以

    • 列表

      • 无序表

        • 在段首加星号(*)

        • 在段加加号 (+)

        • 在段首加减号（-）

      • 有序表

        1. 用数字接着一个英文句点

        2. 在内部不会自增有点坑人

        3. 的确坑人

          • 代码块(缩进4个空格或是1个制表符就可以)undefined

      val s = “hello Markdown”
      println( s )

      • 分割线

        • 三个以上的星号

        • 三个以上的减号

        • 三个以上的底线

          —— ——— ——— ————

      • 链接

        • 链接文字都是用方括号[ ]来标记的

          This is an example inline link.

      [foo]: http://example.com/  "Optional Title Here"

      [foo]: http://example.com/  'Optional Title Here'

      [foo]: http://example.com/  (Optional Title Here)
* 加粗
   - 使用两个星号（**）和底线（_）最为标记强调字词的符号。

       **星号加粗效果**

       __底线加粗效果__

* 斜体
   - 使用一个星号（**）和底线（_）最为标记强调字词的符号。

          *星号斜体效果*

           _底线斜体效果_
 *  图片（在纯文本应用中添加图片是有一定的难度的，唯一的方法就是吧图片转换位文字的表示方法，这里面可以用url来代替,想要设置宽和高用HTML的img标签。一般的办法就是上传按倒图床上面，通过图床提供的图片地址进行链接然后插入图片，也可以用其他的方法进行处理）

     - 一个惊叹号

     - 接着一个方括号，里面放上图片的替代文字

     - 接着一个普通括号，里面放上图片的网址，最后还可以用引号包住并加上 选择性的 'title' 文字

     ![清纯小美女](http://f.hiphotos.baidu.com/image/h%3D360/sign=b815d604b7b7d0a264c9029bfbee760d/b2de9c82d158ccbf79a00f8c1cd8bc3eb1354163.jpg)
 * 反斜杠（可以利用反斜杠来插入一些在语法中有其它意义的符号）
